# Future Work

SalesSense AI has a solid analytics foundation, and several enhancements could extend it into a more production-like system.

## Backend Integration

- Add a live API for the dashboard instead of relying only on static files.
- Serve metrics, charts, and ROI data through endpoints.
- Support authenticated access for different user roles.

## Real-Time Analytics

- Stream fresh data into MySQL.
- Recompute metrics and dashboards automatically.
- Add scheduled or event-driven refresh jobs.

## More Datasets

- Expand beyond the current two datasets.
- Add domain variants such as store-level, regional, and seasonal retail data.
- Build a dataset registry so new sources can be onboarded with minimal changes.

## Advanced ML Models

- Add tuned gradient boosting variants.
- Explore stacking or blending ensembles.
- Add time-series forecasting models for trend projection.
- Improve interpretability with SHAP or similar explanation methods.

## Product Enhancements

- Add alerting for anomalies and drift.
- Provide downloadable executive reports.
- Add trend comparisons across datasets.
- Introduce interactive filters for deeper analysis.
